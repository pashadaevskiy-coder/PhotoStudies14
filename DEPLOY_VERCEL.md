# Инструкция по деплою на Vercel

## 1. Регистрация
1. Зайдите на https://vercel.com/
2. Зарегистрируйтесь бесплатно (Google аккаунт)
3. Подтвердите email

## 2. Подготовка проекта
1. Убедитесь, что у вас есть файлы:
   - app.py
   - requirements.txt
   - vercel.json
   - templates/ (папка со всеми шаблонами)
   - static/ (папка со статическими файлами)

## 3. Импорт проекта
1. В Vercel нажмите "New Project"
2. Выберите "Import Git Repository"
3. Выберите ваш репозиторий
4. Нажмите "Import"

## 4. Настройка проекта
1. В настройках проекта:
   - Framework Preset: Other
   - Build Command: `pip install -r requirements.txt`
   - Output Directory: (оставить пустым)
   - Install Command: `pip install -r requirements.txt`

## 5. Переменные окружения
1. В разделе "Environment Variables" добавьте:
   - FLASK_ENV: production
   - DATABASE_URL: sqlite:///photo_studios.db
   - SECRET_KEY: ваш_секретный_ключ

## 6. Деплой
1. Нажмите "Deploy"
2. Дождитесь завершения сборки
3. После успешного деплоя сайт будет доступен по Vercel URL

## 7. Инициализация базы данных
1. После деплоя нужно создать базу данных
2. Можно использовать API endpoint: `https://ваш-сайт.vercel.app/api/init-db`
3. Или создать отдельный скрипт для инициализации

## Важно!
- Vercel имеет ограничения на бесплатном тарифе
- База данных SQLite будет храниться в файле
- Для работы формы записи нужно настроить переменные окружения
- Если возникнут проблемы с правами доступа, можно попробовать использовать временные папки
